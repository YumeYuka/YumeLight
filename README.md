# YumeLight-vscode-theme

> 一个 VS Code 主题   

![alt text](./images/image.png)
## 特性

*   白色和粉色的配色方案。
*   为多种语言提供语法高亮。

## 安装

1.  在 VS Code 的扩展商店中搜索 `YumeLight` 并安装。
2.  在 **文件 > 首选项 > 主题 > 颜色主题** 中选择 `Yume Light`。

## 开发

在本地开发时，需要合并 token 文件：

```sh
node themes/merge-theme.js
```
